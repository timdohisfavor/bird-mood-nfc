# Bird Call Audio Assets

Place real bird call files in this folder using the bird id as the filename.

MVP filenames:

- `sparrow.mp3` - 麻雀
- `egret.mp3` - 大白鹭
- `zebra-dove.mp3` - 珠颈斑鸠
- `moorhen.mp3` - 红嘴鸥
- `white-headed-duck.mp3` - 白头鹎

Recommended format:

- MP3
- 1-3 seconds
- 96-160 kbps
- Mono preferred
- No long silence at the beginning
- Gentle fade out at the end
